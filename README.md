# 张和贵 GitHub Pages 学术主页

这是一个可直接部署到 GitHub Pages 的中英双语静态学术主页。

## 文件结构

- `index.html`：主页结构、双语内容与论文链接
- `styles.css`：页面视觉样式
- `script.js`：语言切换、滚动显现动画与年份更新

## 部署到 GitHub Pages

1. 在 GitHub 新建仓库：`HeguiZhang.github.io`
2. 将本目录下全部文件上传到仓库根目录
3. 提交后，GitHub Pages 会自动发布
4. 访问地址通常为：`https://heguizhang.github.io/`

## 当前特性

- 中文 / 英文切换
- Google Scholar 入口
- 代表论文原文链接
- 论文作者身份备注（如第一作者、通讯作者）
